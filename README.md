# Usage

1. Сбросить базу данных
	```bash 
	pnpm run reset
	```
2. Собрать проект
	```bash 
	pnpm run build
	```
3. Запустить сервер
	```bash
	pnpm run start
	```
4. Остановить сервер когда все зарегались
	```bash
	ctrl + c
	```
5. Сгенерировать группы
	```bash
	pnpm run gen
	```
6. Отправить информацию о группах каждому юзеру
	```bash
	pnpm run send-group-info
	```
7. Открыть страницу с таблицей
	```bash
	pnpm run open-html
	```
